🔧 TROUBLESHOOTING
================================================================================

ISSUE: "Node version >=16 required"
SOLUTION:
  $ nvm use
  $ node --version

ISSUE: "Cannot find module 'nodemon'"
SOLUTION:
  $ npm install --save-dev nodemon

ISSUE: "Port 10000 already in use"
SOLUTION (macOS/Linux):
  $ lsof -ti:10000 | xargs kill -9
  $ npm run dev

SOLUTION (Windows, PowerShell):
  $ netstat -ano | findstr :10000
  $ taskkill /PID <PID> /F

ISSUE: "Cannot connect to database"
SOLUTION:
  1. Check DATABASE_URL in .env is correct
  2. Verify PostgreSQL is running
  3. Check credentials are valid
  4. Run: npm run setup

ISSUE: "Environment validation failed"
SOLUTION:
  1. Check all required vars in .env
  2. Ensure ENCRYPTION_KEY is 16+ chars
  3. Verify PORT and DB_PORT are numbers
  4. Run: npm run dev (will show detailed errors)
